macOS Installation & First-Time Setup

SnapRewrite Installation & Setup Guide

This guide follows the real first-run flow on macOS: download the installer, complete installation, launch SnapRewrite from Applications, open Settings from the menu bar, grant Accessibility permission, choose a trigger mode, and configure AI models. Each step includes a matching screenshot so you can compare what you see on your screen as you go.

Download Latest Installer Back to Home FAQ (English)

Before You Start

It is best to follow the steps in order. During first launch, pay attention to any macOS permission dialogs. Once setup is complete, you can trigger SnapRewrite from any app where text selection is available, using the trigger mode you configured.

01

Download the latest installer

Get the newest SnapRewrite macOS package from GitHub Releases.

Entry: GitHub Releases Target: latest installer package
  1. Open the SnapRewrite Releases page.
  2. Find the latest release and locate the macOS installer package.
  3. Click the installer file and wait for the download to finish.
If multiple files are listed, choose the installer package from the latest release that is meant for macOS.
Download the newest release

On the Releases page, open the latest release and download the macOS installer package.

02

Open the installer and complete installation

Double-click the downloaded file and finish the macOS installation flow.

Action: double-click installer System: macOS Installer
  1. Locate the downloaded installer in your browser download list or the Downloads folder.
  2. Double-click the file to launch the macOS installer.
  3. Follow the installation wizard until you see the installation completed message.
If macOS shows a security dialog, follow the system prompt. If needed, you can allow the installer in System Settings > Privacy & Security.
Follow the installer flow

Double-click the downloaded file and continue through the standard macOS installation steps.

03

Launch SnapRewrite from Applications

After installation, open SnapRewrite from Applications for the first time.

Location: Applications Result: icon appears in the menu bar
  1. Open the Applications folder.
  2. Find SnapRewrite and double-click it.
  3. After launch, the SnapRewrite icon will appear in the macOS menu bar.
It is normal if no large main window appears. SnapRewrite stays available from the menu bar after launch.
Launch the app and check the menu bar

Once SnapRewrite is open, you will see its icon in the macOS menu bar, which means the app is running.

04

Open the Settings panel

Use the SnapRewrite icon in the menu bar to open the app settings.

Entry: menu bar Menu item: Settings
  1. Click the SnapRewrite logo in the menu bar.
  2. Select Settings from the dropdown menu.
  3. Confirm that the settings panel is now open.
If the settings panel is not open yet, click the menu bar icon once and choose Settings.
Open Settings from the menu

Click the SnapRewrite icon in the menu bar, then select Settings from the dropdown menu.

05

Open the Accessibility permission flow in General

SnapRewrite needs Accessibility permission so it can read selected text and trigger actions correctly.

Path: Settings > General Button: Grant Permission
  1. In the settings panel, switch to the General tab.
  2. Find the Accessibility permission section.
  3. Click Grant Permission to open the macOS authorization settings.
After you click Grant Permission, macOS will take you directly to the related system permission page.
Click the permission button in General

Open the General tab and click Grant Permission in the Accessibility section.

06

Authorize SnapRewrite in macOS

Enable SnapRewrite in the system permission panel, then return to SnapRewrite Settings.

System panel: Privacy & Security Return to: SnapRewrite Settings
  1. In the macOS permission settings, locate SnapRewrite.
  2. Turn on the permission toggle for SnapRewrite and confirm if macOS asks again.
  3. Return to the SnapRewrite Settings panel.
If the permission state does not refresh right away, return to SnapRewrite and wait a moment, or reopen the settings panel.
Turn on the system permission

In macOS Accessibility settings, locate SnapRewrite and enable its toggle.

07

Choose your preferred Trigger Mode

In the Trigger section, pick the interaction style that best fits your workflow.

Path: Settings > Trigger Options: Drag / Double-click / Shortcut
  1. Open the Trigger tab in the settings panel.
  2. Choose your preferred trigger mode: Drag, Double-click, or Shortcut.
  3. If you choose Shortcut, configure the keyboard shortcut you want to use.
If you want a consistent interaction across apps, setting a shortcut is often the easiest option.
Pick the trigger mode you prefer

Choose between Drag, Double-click, or Shortcut, then finish the related setup.

08

Configure AI Models if needed

If you want to use your own model setup, finish that configuration in the Models section.

Path: Settings > Models Status: Current Model is the active model
  1. Go to Settings > Models.
  2. Enter or select the AI model configuration you want to use.
  3. After setup, confirm that Current Model shows the model that is currently active.
If you use your own model setup, complete it here and make sure the active model shown is the one you expect.
Confirm the Current Model

After setup, check Current Model to make sure the currently active model is the one you want to use.

09

Close Settings and run a quick real-world test

Finish setup by selecting text and triggering SnapRewrite with the method you just configured.

Final step: close Settings Verification: trigger on selected text
  1. Close the Settings panel to finish configuration.
  2. Select a piece of text in any app.
  3. Use your configured trigger method, such as drag, double-click, or shortcut, to confirm SnapRewrite works as expected.
If nothing happens here, recheck Step 6 for Accessibility permission and Step 7 for your trigger settings.
Use it on selected text right away

Once setup is complete, select text in any supported app and trigger SnapRewrite using the method you configured.

Helpful Tips

  • If you work with text across many apps, choosing one trigger mode you can remember easily will save time.
  • After Accessibility is granted, SnapRewrite should work in most text-selection scenarios right away.
  • If it stops responding later, check the permission status in General first.

Preferred macOS Wording

  • Use Applications for the app folder.
  • Use menu bar for the top macOS bar.
  • Use Accessibility exactly as the system names the permission.

Ready to Use

After the steps above, SnapRewrite is ready in any app that supports text selection. Select text, trigger the app with the method you configured in Trigger, and start rewriting, polishing, translating, or running other AI actions.
