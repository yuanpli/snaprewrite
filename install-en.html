SnapRewrite Installation Guide (English)

For the website trial package (.pkg). Includes Gatekeeper allowlisting, Accessibility permission setup, and first-time Keychain prompt guidance.

Back to Home FAQ (English)

1) Download and Install the Trial Build

  1. Download: SnapRewrite-1.0.0-universal.pkg
  2. Double-click the package and finish the installer flow.
  3. Default install path: /Applications/SnapRewrite.app
Download Trial Build

2) Allow Local Installation (Gatekeeper)

  1. If macOS blocks launch, open System Settings > Privacy & Security.
  2. Find the SnapRewrite blocked message and click “Open Anyway”.
  3. Open the app again, or right-click in Finder and choose “Open”.

3) Enable Accessibility Permission (Required)

  1. Open SnapRewrite and go to Settings > General.
  2. Click “Grant Permission…” to jump to Accessibility settings.
  3. Enable SnapRewrite and confirm the toggle.
  4. Return to the app and confirm status shows “Permission granted”.

4) First Keychain Prompt (Expected)

Important: this is not an error and not a risk warning about the app itself. It is a standard macOS security check.
SnapRewrite uses Keychain for sensitive values (such as API keys and trial-related secure values). On first access, macOS asks for your confirmation. After entering your login password, choosing “Always Allow” is recommended to reduce repeated prompts.
