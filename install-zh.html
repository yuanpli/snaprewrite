SnapRewrite 安装说明（中文）

适用于官网体验版安装包（.pkg）。包含本地安装放行（Gatekeeper）、辅助功能权限开启、以及首次 Keychain 提示说明。

返回首页 FAQ（中文）

1) 下载体验版并安装

  1. 下载：SnapRewrite-1.0.0-universal.pkg
  2. 双击安装包，按向导完成安装。
  3. 默认安装路径：/Applications/SnapRewrite.app
下载体验版

2) 允许本地安装（Gatekeeper）

  1. 若系统提示“无法验证开发者”或“不允许打开”，进入 系统设置 > 隐私与安全性。
  2. 在页面下方找到 SnapRewrite 被拦截提示，点击“仍要打开”。
  3. 返回应用再次打开，或在 Finder 右键应用选择“打开”。

3) 启用辅助功能权限（必须）

  1. 打开 SnapRewrite，进入 设置 > 通用。
  2. 点击“授予权限…”，跳转到系统“辅助功能”设置。
  3. 开启 SnapRewrite 开关并确认。
  4. 回到应用，状态显示“已授予权限”。

4) 首次 Keychain 弹窗（正常）

重点：这不是异常，也不是风险告警，而是 macOS 标准安全校验流程。
SnapRewrite 使用 Keychain 保存敏感信息（如 API Key 与试用相关安全值），首次访问时系统会要求你确认授权。输入登录密码后，建议选择 “Always Allow（始终允许）” 以减少重复弹窗。
