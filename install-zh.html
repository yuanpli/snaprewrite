SnapRewrite 安装与配置指南

这份指南按照实际的 macOS 使用流程整理：从下载安装包、完成安装、从 Applications 打开，到菜单栏设置、辅助功能授权、触发方式与模型配置。每一步都配有对应截图，你可以对照屏幕上的界面快速完成设置。

开始前

建议按照页面顺序完成。首次启动时，请留意 macOS 的系统授权弹窗。配置完成后，你就可以在任意支持文本选择的应用中，通过你设置的触发方式来调用 SnapRewrite。

01

下载最新安装包

从 GitHub Releases 获取最新版本的 SnapRewrite macOS 安装文件。

入口：GitHub Releases 目标文件：最新版安装包
  1. 打开 SnapRewrite Releases 页面。
  2. 在最新版本中找到适用于 macOS 的安装包。
  3. 点击下载安装文件，等待下载完成。
如果你看到多个文件，请优先下载最新发布版本中适用于 macOS 的安装文件。
找到最新版本并下载

在 Releases 页面中找到最新发布版本，点击对应的 macOS 安装文件开始下载。

02

打开安装文件并完成安装

双击下载好的安装文件，按照 macOS 安装向导完成安装流程。

操作：双击安装文件 系统：macOS Installer
  1. 在浏览器下载列表或 Downloads 文件夹中找到刚下载的安装文件。
  2. 双击该文件，启动 macOS 安装程序。
  3. 按安装向导完成安装，直到看到安装成功提示。
如果 macOS 弹出安全提示，请按系统提示继续；如有需要，可在 System Settings > Privacy & Security 中允许打开。
按安装向导继续

双击安装文件后，按照标准的 macOS 安装流程点击继续并完成安装。

03

从 Applications 打开 SnapRewrite

安装完成后，从 Applications 中找到 SnapRewrite 并首次启动。

位置：Applications 结果：菜单栏出现图标
  1. 打开 Applications 文件夹。
  2. 找到 SnapRewrite 应用并双击打开。
  3. 启动后，程序图标会出现在系统菜单栏（menu bar）中。
首次打开后，应用可能不会弹出单独主窗口，这是正常的，SnapRewrite 会常驻在菜单栏中供你使用。
打开应用并留意菜单栏

启动 SnapRewrite 后，你会在 macOS 的菜单栏中看到它的图标，这表示应用已经运行。

04

打开 Settings 配置面板

通过菜单栏中的 SnapRewrite 图标进入应用设置。

入口：menu bar 菜单项：Settings
  1. 点击菜单栏中的 SnapRewrite logo。
  2. 在下拉菜单中选择 Settings。
  3. 确认配置面板已打开，准备开始首次设置。
如果你还没有看到设置面板，点击菜单栏图标后选择 Settings 即可进入。
从菜单中进入 Settings

点击菜单栏中的 SnapRewrite 图标后，选择 Settings 打开配置面板。

05

在 General 中打开 Accessibility 授权入口

SnapRewrite 需要 Accessibility 权限，才能读取选中文本并执行触发操作。

路径：Settings > General 按钮：Grant Permission
  1. 在设置面板中切换到 General 标签页。
  2. 找到 Accessibility 权限区域。
  3. 点击 Grant Permission，系统会跳转到 macOS 授权设置页面。
点击 Grant Permission 后，macOS 会带你进入对应的系统权限页面。
在 General 中点击授权按钮

进入 General 后，在 Accessibility 区域点击 Grant Permission。

06

在 macOS 中授权 SnapRewrite

启用系统辅助功能授权后，返回 SnapRewrite Settings 继续后续配置。

系统面板：Privacy & Security 返回：SnapRewrite Settings
  1. 在 macOS 打开的权限设置页中找到 SnapRewrite。
  2. 打开 SnapRewrite 对应的授权开关，如系统要求请确认。
  3. 完成后返回 SnapRewrite Settings 页面。
如果权限状态没有立刻刷新，返回 SnapRewrite 后稍等片刻，或重新打开设置页确认是否生效。
在系统设置中打开权限

在 macOS 的 Accessibility 列表中找到 SnapRewrite，并开启它的授权开关。

07

配置你喜欢的 Trigger Mode

在 Trigger 页面选择最适合你工作流的触发方式。

路径：Settings > Trigger 选项：Drag / Double-click / Shortcut
  1. 在设置面板中进入 Trigger 标签页。
  2. 根据你的习惯选择触发模式：Drag、Double-click 或 Shortcut。
  3. 如果你选择 Shortcut，继续设置你偏好的快捷键组合。
如果你希望在不同应用里保持一致的触发习惯，通常设置一个自己顺手的快捷键会更方便。
选择你最顺手的触发方式

你可以在这里选择 Drag、Double-click 或 Shortcut，并完成对应设置。

08

按需配置 AI Models

如果你需要自定义模型服务或模型列表，可以在 Models 页面完成配置。

路径：Settings > Models 状态：Current Model 为当前生效模型
  1. 进入 Settings > Models。
  2. 根据你的需求填写或选择要使用的 AI 模型配置。
  3. 配置完成后，确认 Current Model 显示的是当前生效的 active 模型。
如果你使用自定义模型服务，请在这里完成配置，并确认当前生效模型显示正确。
确认 Current Model

配置完成后，查看 Current Model，确认当前激活的模型就是你想使用的那一个。

09

关闭 Settings 并进行一次实际测试

完成设置后，立即用一段已选中的文本测试你的触发方式是否生效。

最后一步：关闭 Settings 验证：选中文本后触发
  1. 关闭 Settings 面板，保存当前配置。
  2. 在任意应用中选择一段文字。
  3. 按下你设置好的触发方式，例如拖拽、双击或快捷键，验证 SnapRewrite 已可以正常调用。
如果这一步没有反应，请先检查第 6 步的权限是否已开启，以及第 7 步的触发方式是否设置正确。
选中文本后直接触发

设置完成后，在任意应用中选中一段文字，再用你设定的方式触发 SnapRewrite 即可开始使用。

使用小提示

  • 如果你经常跨应用处理文字，建议优先设置自己最顺手的触发方式。
  • 首次完成授权后，SnapRewrite 通常就可以在大多数文本选择场景中直接使用。
  • 若后续没有响应，优先回到 General 检查权限状态。

macOS 术语统一

  • 应用目录建议统一写作 Applications。
  • 系统顶部区域统一写作 menu bar。
  • 权限名称统一为 Accessibility，与系统实际文案保持一致。

准备开始

完成以上设置后，你就可以在支持文本选择的场景中使用 SnapRewrite。选中文本，再用你在 Trigger 中设定的方式触发，即可开始改写、润色、翻译或其它 AI 操作。
