SnapRewrite FAQ (English)

Quick answers for website trial users: install allowlisting, permission setup, Keychain prompt, privacy, model usage, and support.

Back to Home Installation Guide Privacy Policy Terms of Service
1) Which apps are supported?
Most editable text fields are supported, including email, docs, chat apps, and note tools.
2) Is Free version limited?
Free includes daily quota and core features, enough to run the complete workflow.
3) Can I connect my own model?
Yes. Current options include Zhipu (glm-4-flash), OpenAI (gpt-4.1), Groq (llama-3.3-70b-versatile), and Gemini (gemini-3-flash-preview).
4) Where does my data go?
Config and keys are stored locally. Requests are only sent when you trigger an action.
5) How do I install the website trial build?
Download the pkg and follow the installer flow. If launch is blocked, go to System Settings > Privacy & Security and click “Open Anyway”. See Installation Guide.
6) Why is Accessibility permission required?
SnapRewrite needs it to detect selected text in other apps and perform in-place replacement. Without this permission, cross-app rewrite flow will not work.
7) Is the Keychain password prompt expected?
Yes. SnapRewrite stores sensitive data in Keychain. On first access, macOS asks you to confirm. Enter your login password and choose “Always Allow” to reduce repeated prompts.
8) What's the difference between trial build and App Store version?
The website build is a full-feature trial for a limited period, then it falls back to free mode. The App Store version is distributed and managed through Apple, including purchase/restore flows.
9) Does the app upload all my documents automatically?
No. It only processes the currently selected text when you explicitly trigger an action.
10) Where is my API key stored?
User-provided API keys are stored in macOS Keychain, not plain text settings files. You can edit or remove them anytime in Settings.
11) How can I report issues or suggestions?
Please email pingldudu@gmail.com with steps to reproduce, logs/screenshots, and your macOS version if possible.
