SnapRewrite 常见问题（中文）

这里整理了官网体验版高频问题：安装放行、权限配置、Keychain 提示、隐私与模型能力等。

1) 支持哪些应用？
支持绝大多数可编辑文本区域，例如邮件、文档、聊天工具和笔记应用。
2) 免费版有限制吗？
免费版提供每日额度和核心功能，适合完整体验真实工作流。
3) 是否支持自定义模型？
支持。当前可选模型包括：智谱（glm-4-flash）、OpenAI（gpt-4.1）、Groq（llama-3.3-70b-versatile）、Gemini（gemini-3-flash-preview）。
4) 数据会上传到哪里？
配置和密钥保存在本地，仅在你手动触发时向目标模型发送请求。
5) 官网体验版如何安装？
请先下载安装包并按向导完成安装。若被系统拦截，请到 系统设置 > 隐私与安全性 点击“仍要打开”。详细步骤见 安装说明。
6) 为什么必须开启“辅助功能”权限？
SnapRewrite 需要在其他应用内识别选中文本并进行原位替换，这属于 macOS 受控能力。未授权时只能看到界面，无法执行核心跨应用改写流程。
7) 首次弹出 Keychain 密码是正常的吗？
是正常现象。应用会把 API Key 等敏感信息存放在 Keychain。首次访问时系统要求你确认授权，建议输入登录密码后选择 “Always Allow”。
8) 体验版和 App Store 正式版有什么区别？
官网体验版提供完整功能限时体验，到期后回到免费功能模式；正式版通过 App Store 分发与维护，购买与恢复权益由 Apple 机制处理。
9) 数据会被上传吗？会读取我全部文档吗？
不会自动扫描全部文档。仅在你主动触发时处理当前选中的文本片段，并发送到你选择的模型服务商进行生成。
10) API Key 保存在哪里？
用户自填 API Key 存储在 macOS Keychain，不以明文写入普通配置文件。你可以随时在设置中修改或删除。
11) 遇到问题如何反馈？
欢迎发邮件到 pingldudu@gmail.com，请尽量附上问题描述、复现步骤和系统版本信息。
12) 我在 App Store 版已购买 Pro，能迁移到官网独立版继续使用吗？
可以。我们建议已购买 Pro 的用户迁移到官网独立版继续使用：App Store 版受沙盒权限约束，部分跨应用能力在使用上会略有不便。迁移后可继续保留 Pro 权益（无需重复购买）。如你在迁移过程中遇到问题，请联系 pingldudu@gmail.com 获取协助。
